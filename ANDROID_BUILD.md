# Android Build Instructions

## First-time Setup

1. Generate Android native folder:
   ```powershell
   npx expo prebuild --platform android --clean
   ```

2. Configure Android SDK:
   ```powershell
   .\scripts\setup-android.ps1
   ```

## Building APK

### Debug Build (with Metro dev server)
```powershell
cd android
.\gradlew.bat assembleDebug
cd ..
adb install android\app\build\outputs\apk\debug\app-debug.apk
```

### Release Build (standalone, no Metro needed)
```powershell
cd android
.\gradlew.bat assembleRelease
cd ..
adb install -r android\app\build\outputs\apk\release\app-release.apk
```

## Troubleshooting

### "SDK location not found" error
Run the setup script:
```powershell
.\scripts\setup-android.ps1
```

### Clean build
```powershell
cd android
.\gradlew.bat clean
cd ..
```

## Configuration Files

- `app.json` - Expo configuration (package name, version code, SDK versions)
- `android/local.properties` - SDK path (auto-generated by setup script, gitignored)
- `android/app/build.gradle` - Build configuration
- `android/gradle.properties` - Gradle settings
